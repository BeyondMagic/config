This is the "Yellow Dot cursors, left-handed edition" cursor theme for XFree 4.3+
========================================================

This cursor set was based on the Yellow Dot cursor set
by Telex: http://kde-look.org/content/show.php?content=5475
which was in turn based on the Red Dot cursor set by uga:
http://kde-look.org/content/show.php?content=4805

========================================================

a)How to install:

Just type "make install"


b) How to regenerate the cursors after a change:

Type "make" (or make install, if you want to install them straight)

-

See more documentation at the XCursors_HowTo directory.
